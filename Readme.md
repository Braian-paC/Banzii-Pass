Olá esse projeto se chama "Banzii Pass", e a ideia dele é ser um gerador de senhas fortes e randômicas com uma interface e mecânicas atraentes e funcionais.
Ideia de mecânica base: Você pode escolher se você quer gerar uma senha totalmente aleatória ou inverter caractéres aleatórios de uma senha. OBS: É necessário escolher o número de caractéres.
