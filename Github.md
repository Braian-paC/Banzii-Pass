Esse é o primeiro projeto que estou criando no Github, então vou guardar algumas anotações aqui:

git add .
git status
git commit -m "nome da alteração"
git push origin main
git checkout -b "new-branch"
git checkout branch
git merge branch

cd projeto
git pull
fork = copiar o projeto de outra pessoa para o seu repositório
pull request = literal do nome, requere o acesso do dono do projeto para realizar o pull
